# Support

Epiktetos is an open-source WordPress theme. Support is provided through project documentation and GitHub issues.

## Documentation

Read the documentation first:

[https://docs.mcorucu.com/epiktetos-theme/](https://docs.mcorucu.com/epiktetos-theme/)

## Issues

Use the issue tracker for reproducible bugs, accessibility problems, compatibility issues, and documentation corrections:

[https://github.com/mcorucu/epiktetos-wordpress-theme/issues](https://github.com/mcorucu/epiktetos-wordpress-theme/issues)

When opening an issue, include:

- WordPress version.
- PHP version.
- Theme version.
- Browser and operating system when relevant.
- Steps to reproduce.
- Expected and actual behavior.

## Not Covered

General WordPress administration, hosting configuration, plugin conflicts outside the theme's control, and custom site builds may fall outside project support.
