# Security Policy

## Supported Versions

Security fixes are considered for the current public release of Epiktetos.

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |

## Reporting a Vulnerability

Please do not open a public issue for a suspected security vulnerability.

Report security concerns through GitHub's private vulnerability reporting if available, or contact the maintainer directly with:

- A clear description of the issue.
- Steps to reproduce.
- Affected WordPress, PHP, and theme versions.
- Any proof-of-concept details needed to validate the report.

Reports will be reviewed as soon as possible. Confirmed issues will be handled with an appropriate fix and release note.

## Scope

This policy covers vulnerabilities in the Epiktetos theme code. Issues in WordPress core, plugins, hosting environments, or third-party services should be reported to the relevant maintainers.
